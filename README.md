# WordPressテーマ化ポートフォリオ（Coffee Site）

公開URL：https://practice.yuukimedia.com  

## 概要
書籍の模写コーディングで作成した静的HTMLサイトをベースに、WordPressテーマ化して運用を想定した構成に拡張しました。  
未経験者として「既存コードを読み、崩さずに改修する」ことを意識しています。

## 使用技術
- WordPress（オリジナルテーマ化）
- PHP（テンプレート作成・ループ）
- HTML / CSS
- JavaScript

## 実装内容
- 静的HTML → WordPressテーマ化（テンプレート分割）
- 投稿一覧（ブログ）表示（ループ）
- カテゴリーページの新規設置
- 404ページの新規設置
- ページネーション実装

## 工夫した点
- 既存デザインを崩さないようにテンプレート分割とCSS構成を整理
- 更新作業を想定し、表示崩れや誤記が起きにくい構造を意識
- WordPressのテンプレート階層に沿って「必要最低限のページ」を整備（カテゴリ／404など）

## アピールポイント
- 「静的→WP化＋拡張」の流れ（更新・運用案件想定）
- カテゴリー／404など、実務で必要になりやすいページの整備
- ループ処理・テンプレート分割の理解
